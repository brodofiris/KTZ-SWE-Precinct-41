# 🚆 Панель управления поездом (Train Telemetry Dashboard)

## 📖 О проекте
Данный проект представляет собой интерактивную телеметрическую панель для мониторинга систем поезда в режиме реального времени. 
Ссылка на [видео демонстрацию:](https://youtu.be/a2p9xESk-LE)

**Ключевая особенность интерфейса** — строгая фиксация всех элементов управления и показателей на одном экране (Single-Screen UI) без использования прокрутки (scroll). Этот архитектурный выбор продиктован строгими стандартами эргономики и безопасности кабин локомотивов: машинисты и операторы не имеют возможности отвлекаться на скроллинг или навигацию по сложным меню во время движения состава. Все критически важные данные доступны с одного взгляда.

## 🛠 Стек технологий
* **Backend:** Python, FastAPI, WebSockets
* **Frontend:** HTML, CSS, JavaScript
* **Визуализация и карты:** Leaflet.js, Chart.js

## 🚀 Как запустить локально

Чтобы развернуть проект на своей машине, выполните следующие шаги в указанном порядке:

### 0. Создание .env перемнной
Необходимо создать файл .env и записать в него следующее значение
```
SECRET_KEY=super_secret_train_key_12345
```

### 1. Создание администратора
Перед запуском системы необходимо создать первичную учетную запись с правами администратора для доступа к системе управления.
*(Укажите здесь команду или скрипт, который вы используете для этого, например: `python create_admin.py`)*

### 2. Запуск симуляции телеметрии
Для имитации работы систем поезда (показатели датчиков, скорость, координаты) необходимо запустить скрипт генерации данных. Откройте терминал и выполните:
```bash
python dummy_data.py
```
### 3. Запуск веб-сервера
В новом окне терминала запустите основное приложение FastAPI с помощью Uvicorn. Сервер обеспечит работу API и WebSocket соединений для передачи данных на фронтенд в реальном времени:
```bash
uvicorn main:app --reload
```
После этого откройте браузер и перейдите по адресу: http://localhost:8000 (или по тому порту, который указан в ваших настройках).
